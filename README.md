# 天气播报插件 
定时获取指定地区天气并推送到群聊，同时支持手动查询和 LLM 工具调用。本插件通过高德地图的 Web 服务 API 抓取实时的天气数据及行政区划信息。

---

## ✨ 功能特性

* **定时群组播报**: 每日定时通过 LLM 生成符合麦麦人设的天气概况提醒并推送到指定群聊。
* **手动指令查询**: 用户可通过简单指令手动查询特定地区的实时天气以及多日预报。
* **LLM 智能工具**: 为大语言模型提供函数调用支持，使麦麦能在自然对话中解答气温、降雨等相关问题。

---

## 🔑 准备工作

本插件需要配置高德地图 API 才能正常工作。

* 您需要前往高德开放平台申请对应的 API 密钥。
* 申请地址为：`https://console.amap.com/dev/key/app`。

---

## ⚙️ 配置指南

在机器人的插件配置界面中，您会看到以下模块供自定义调整：

### 1. 插件基础配置

* **是否启用插件**: 控制插件的总体开关，默认为启用 (`True`)。
* **配置版本**: 当前版本号 `2.0.0`。

### 2. 天气 API

* **amap_key**: 请在此处填入您申请到的高德地图 API 密钥。未配置时服务将无法正常请求。

### 3. 群聊推送

* **群聊推送列表**: 每行设置一组推送规则，格式必须为“群号 地点名称”，中间以空格分隔。地点支持精确到省、市、区级别。
* **配置示例**: `123456789 河北省`。

### 4. 定时任务

* **是否启用定时推送**: 开关每日的播报功能（若关闭，手动指令和 LLM 查询工具仍然可用）。
* **每日播报时间**: 设置触发播报的时间，格式为 `HH:MM`，默认值为 `08:00`。

### 5. 播报

* **播报内容最大字数**: 限制大模型生成的播报文本长度，默认为 `200` 字以内。

### 6. 查询工具

* **是否启用天气查询工具**: 控制是否允许 LLM 使用天气查询能力，默认为启用。
* **默认返回未来几天预报**: 设置默认返回的天数，允许范围为 1 到 4，默认值为 `3`。

---

## 🚀 使用说明

### 手动查询指令

在支持的群聊或私聊中输入以下格式的命令即可查询：

* `/天气 <地点名称>`
* `/weather <地点名称>`

**示例**:

> `/天气 北京`
> 这将返回北京的实时天气数据（天气状况、温度、风向风力、湿度、更新时间）以及未来几天的预报概览。

### 对话中的智能查询

如果启用了“查询工具”，用户无需输入特定指令，直接在聊天中询问（例如：“北京明天会下雨吗？” 或 “石家庄现在的气温是多少？”）。LLM 将自动提取“城市”及需要的“天数”，并调用相关数据回复。
